# 接口文档
本文档定义了接口格式
* 数据传输格式均为`JSON`  
* 接口请求方式均为`POST`
* 返回值总是包含以下两个字段  

|   字段名   | 变量类型 | 何时存在 | 说明    |
|        :-:|      :-:|       :-:|      :-:|
|`state`    | bool    | 总是     | 是否成功 |
|`error`    | string  | 失败时   | 错误消息 |
